## Rules to solve Pattern based problems:

1. Use nexted loops
2. Number of times the outer loop runs = Number of rows in the pattern
3. Number of times the inner loop runs = Tells how many columns are there
4. Print within the loops
5. Formulate the inner loop variable(j) in terms of the outer loop variable(i) and the input(n) i.e express j as f(i,n)
